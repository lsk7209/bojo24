# 지원 대상 영역 Gemini 보완 가이드

## 🎯 기능 개요

특정 보조금 ID에 대해서만 **지원 대상** 영역을 Gemini로 보완하여 **100~150자**로 가독성 있게 정리합니다.

## ⚙️ 설정 방법

### Vercel 환경 변수 설정

1. Vercel 대시보드 접속
2. 프로젝트 선택 → Settings → Environment Variables
3. 다음 환경 변수 추가:

```
GEMINI_ENHANCEMENT_ALLOWED_IDS=305000000283
```

**설명:**
- `305000000283` - (국가유공자)인플루엔자 백신 및 접종 지원
- 쉼표로 구분하여 여러 ID 추가 가능: `305000000283,305000000284`

4. 재배포 실행

## 📋 작동 방식

### 조건
- 환경 변수 `GEMINI_ENHANCEMENT_ALLOWED_IDS`에 보조금 ID가 포함되어 있어야 함
- 지원 대상 내용이 150자 미만이어야 함
- `GEMINI_API_KEY`가 설정되어 있어야 함

### 결과
- 공공데이터: "관내 지역주민 중 국가유공자 본인" (18자)
- Gemini 보완 후: 100~150자의 가독성 있는 설명

### 예시

**공공데이터 원본:**
```
관내 지역주민 중 국가유공자 본인
```

**Gemini 보완 후 (100~150자):**
```
서울특별시 동대문구에 거주하시는 국가유공자 본인이라면, 인플루엔자 백신 및 접종 지원 대상이 되실 수 있습니다. 
이 지원은 현재 별도의 나이, 소득 제한 없이 동대문구에 거주하시는 국가유공자 본인을 대상으로 하고 있습니다.
```

## 🔍 확인 방법

1. Vercel 환경 변수 설정
2. 재배포 완료 대기
3. 해당 페이지 접속:
   ```
   https://www.bojo24.kr/benefit/보건·의료/305000000283
   ```
4. **지원 대상** 섹션 확인
   - 100~150자의 상세한 설명이 표시되어야 함
   - 가독성 있고 구체적인 내용

## 📊 현재 상태

- ✅ 지원 대상 영역만 Gemini 보완
- ✅ 100~150자로 제한
- ✅ 특정 보조금 ID만 활성화 가능
- ✅ 다른 섹션은 공공데이터만 사용

## ⚠️ 주의사항

1. **비용 관리**
   - 해당 보조금 페이지 접속 시에만 Gemini API 호출
   - 다른 보조금은 공공데이터만 사용 (비용 없음)

2. **재배포 필요**
   - 환경 변수 변경 후 Vercel 재배포 필요

3. **API 키 확인**
   - `GEMINI_API_KEY`가 Vercel에 설정되어 있어야 함
