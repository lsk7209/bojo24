## 📊 광고 배치 전략 (수익화 최적화)

### 광고 위치

#### 1. 상단 배너 광고
**위치**: 홈페이지, 목록 페이지 상단
```tsx
<BannerAd adSlot="1234567890" />
```
- **효과**: 높은 노출률, 첫 인상
- **주의**: 컨텐츠를 가리지 않도록 배치

#### 2. 인라인 광고 (본문 중간)
**위치**: 상세 페이지 본문 중간
```tsx
<InlineAd adSlot="1234567890" />
```
- **효과**: 높은 클릭률, 자연스러운 배치
- **주의**: 컨텐츠 흐름을 방해하지 않도록

#### 3. 사이드바 광고
**위치**: 데스크톱 사이드바
```tsx
<SidebarAd adSlot="1234567890" />
```
- **효과**: 지속적인 노출
- **주의**: 모바일에서는 숨김 처리

#### 4. 하단 배너 광고
**위치**: 페이지 하단
```tsx
<BannerAd adSlot="1234567890" />
```
- **효과**: 추가 수익 기회
- **주의**: 푸터 위에 배치

### 광고 슬롯 ID 설정

AdSense 승인 후 실제 슬롯 ID로 교체:

```typescript
// 환경 변수 또는 설정 파일에서 관리
const AD_SLOTS = {
  banner: process.env.NEXT_PUBLIC_ADSENSE_SLOT_BANNER || "1234567890",
  inline: process.env.NEXT_PUBLIC_ADSENSE_SLOT_INLINE || "1234567890",
  sidebar: process.env.NEXT_PUBLIC_ADSENSE_SLOT_SIDEBAR || "1234567890",
};
```

## 🎯 수익화 최적화 전략

### 1. 광고 밀도 최적화

**권장 사항**:
- 페이지당 최대 3-4개 광고
- 본문 300자마다 1개 광고
- 모바일: 2-3개, 데스크톱: 3-4개

**현재 배치**:
- 홈페이지: 상단 배너 1개
- 목록 페이지: 상단 인라인 1개
- 상세 페이지: 본문 중간 인라인 1개
- 블로그 포스트: 본문 중간 인라인 1개

### 2. 광고 형식 최적화

**반응형 광고 (권장)**:
```tsx
<AdSenseAd 
  adSlot="1234567890"
  format="auto"
  responsive={true}
/>
```

**장점**:
- 자동 크기 조정
- 높은 수익률
- 사용자 경험 최적화

### 3. 컨텐츠 최적화

**고품질 컨텐츠**:
- AI 기반 요약 (고유성)
- FAQ (검색 최적화)
- 블로그 포스트 (심화 분석)

**키워드 최적화**:
- 보조금명, 카테고리, 지역명
- 자연스러운 키워드 배치
- 롱테일 키워드 활용

### 4. 사용자 경험 최적화

**광고 배치 원칙**:
- 컨텐츠를 가리지 않음
- 자연스러운 흐름
- 명확한 구분 (레이블)

**로딩 최적화**:
- 광고 스크립트 비동기 로딩
- 지연 로딩 (Lazy Loading)
- 에러 처리

1. **명확한 레이블**

   - 접근성 고려 (aria-label)

2. **적절한 배치**
   - 컨텐츠와 명확히 구분
   - 자연스러운 흐름

3. **정기적인 모니터링**

## 🔧 구현 가이드

### 1. AdSense 계정 설정

1. [Google AdSense](https://www.google.com/adsense) 접속
2. 계정 생성 및 사이트 등록
3. Publisher ID 확인: `ca-pub-3050601904412736`
4. 광고 단위 생성 및 슬롯 ID 확인

### 2. 환경 변수 설정

```env
NEXT_PUBLIC_ADSENSE_CLIENT=ca-pub-3050601904412736
NEXT_PUBLIC_ADSENSE_SLOT_BANNER=1234567890
NEXT_PUBLIC_ADSENSE_SLOT_INLINE=2345678901
NEXT_PUBLIC_ADSENSE_SLOT_SIDEBAR=3456789012
```

### 3. 광고 컴포넌트 사용

```tsx

// 본문 중간
<InlineAd adSlot="2345678901" />

// 사이드바
```
